UREN(1) General Commands Manual UREN(1)

NAME

uren — project time tracking

SYNOPSIS

uren [-h]

DESCRIPTION

uren is a project time tracking tool with stopwatch support.
-h
Print usage.

BUILTIN COMMANDS

The key bindings are vi-like. The following commands are supported:
[count] ⟨control-F⟩
Page forward count screens.
[count] ⟨control-B⟩
Page backwards count screens.
[count] ⟨control-D⟩
Scroll forward count half screens.
[count] ⟨control-U⟩
Scroll backwards count half screens.
[count] ⟨control-E⟩
Scroll forward count lines.
[count] ⟨control-Y⟩
Scroll backwards count lines.
[count] j
Move the cursor down count lines.
[count] k
Move the cursor up count lines.
gg Move to the first page.
G Move to the last page.
[count] H
Move to count lines from the top of the current screen.
M
Move to the middle of the current screen.
[count] L
Move to count lines from the bottom of the current screen.
f
Toggle filter on or off.
i
Insert a new entry after the last entry.
O
Insert a new entry before the currently selected line.
I
Alias for O
o
Insert a new entry after the currently selected line.
A
Alias for o.
cc
Edit the current line.
S
Alias for cc.
dd
Delete the current line.
s
Toggle the stopwatch on or off.
q
Quit the application.
The filter, insert and edit form support TAB-completion for the project name. Subsequent use of TAB scrolls through the list of options.

EXIT STATUS

The uren utility exits 0 on success, and >0 if an error occurs.
